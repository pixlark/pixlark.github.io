A Text Editor that I am writing for personal use. I've been a devout Emacs user for a while, but I have SO many issues with it, mostly related to how bloated it is. So, I decided to write my own. It's not fully-featured enough yet for actual use, but it's completely functional. The rendering is done in SDL for simplicity.
