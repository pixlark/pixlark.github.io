Closures

A closure is a technique for implementing lexically scoped name binding in languages with first-class functions.
Essentially, a closure allows you to retain all the information from a function's local scope after the function has terminated.

First-Class Citizen

An element that supports all the operations generally available to other entities.
Usually an element is first-class if it can be passed as an argument, returned from a function, modified, and assigned to a variable.
